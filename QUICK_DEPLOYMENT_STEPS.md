## TROUBLESHOOTING

### "I can't find the Export button"
- Check the top menu
- Look in File menu
- Check top-right corner
- Try right-clicking on the project name

### "GitHub Desktop won't install"
- Make sure you have admin rights
- Try downloading again
- Restart your computer
- Use GitHub.com website directly (more complex)

### "Netlify build failed"
- Check if all files were uploaded to GitHub
- Make sure package.json is included
- Contact Netlify support (they're very helpful)

### "Website shows 404 error"
- Wait 5 minutes for DNS propagation
- Clear browser cache (Ctrl+Shift+R)
- Check Netlify deployment status
