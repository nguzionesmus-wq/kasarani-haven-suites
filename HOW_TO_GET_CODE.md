## Option 1: Download from Figma (Recommended)

### Step 1: Export Your Project
1. In Figma Make (where you are now), look for an **"Export"** or **"Download"** button
2. Click it to download all your website files as a ZIP file
3. The ZIP will contain all your code files

### Step 2: Extract the ZIP
1. Find the downloaded ZIP file (usually in your Downloads folder)
2. Right-click the ZIP file
3. Click **"Extract All..."** or **"Unzip"**
4. Choose a location (like Desktop or Documents)
5. You'll now have a folder with all your website files

### Step 3: You're Ready for GitHub!
Now you have all the files needed for GitHub upload!

## Option 2: Manual Copy (If Export Not Available)

### If you can't find an export button, you can copy files manually:

1. **Create a new folder** on your computer called `kasarani-haven-suites`

2. **Copy each file** from this project:
   - Open each file (App.tsx, etc.)
   - Select all text (Ctrl+A / Cmd+A)
   - Copy (Ctrl+C / Cmd+C)
   - Create same file on your computer
   - Paste (Ctrl+V / Cmd+V)
   - Save

3. **Keep the same folder structure** as shown above

## Quick GitHub Upload Process

### Using GitHub Desktop (Easiest):

1. **Download GitHub Desktop:**
   - Go to https://desktop.github.com
   - Download and install

2. **Sign in to GitHub:**
   - Open GitHub Desktop
   - Sign in with your GitHub account

3. **Create new repository:**
   - File â†’ New Repository
   - Name: `kasarani-haven-suites`
   - Local path: Choose where to create it
   - Click "Create Repository"

4. **Add your files:**
   - Copy all your website files into the new repository folder
   - GitHub Desktop will detect all the files

5. **Commit:**
   - Write a message: "Initial commit - Kasarani Haven Suites website"
   - Click "Commit to main"

6. **Publish:**
   - Click "Publish repository"
   - Choose "Public"
   - Click "Publish repository"

7. **Done!** Your code is now on GitHub!
